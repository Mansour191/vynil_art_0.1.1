# فينيل آرت (Vinyl Art)

مشروع متطور لإدارة وتصفح تصاميم الفينيل، مبني باستخدام Vue.js مع ميزات الذكاء الاصطناعي والتكامل مع ERPNext.

## 🚀 ميزات المشروع
- **إدارة المنتجات**: تكامل كامل مع ERPNext لمزامنة المنتجات والمخزون.
- **توقعات ذكية**: نظام AI مدمج لتوقع المبيعات وإدارة المخزون بشكل استباقي.
- **لوحة تحكم متقدمة**: تتبع مبيعات، تحليلات سلوك العملاء، وتقارير موحدة.
- **تجربة مستخدم غنية**: دعم كامل للغة العربية (RTL) وتصميم عصري متجاوب.
- **أمان عالي**: حماية للمسارات، تعقيم للمدخلات، وتشفير للبيانات الحساسة.

## 🛠️ الخدمات والوحدات الرئيسية

### 1. إدارة المنتجات (`ProductsManager`)
مكون لإدارة المنتجات، يدعم الفلترة المتقدمة، المزامنة مع ERPNext، وعرض إحصائيات المخزون.
- **Props**: `products` (Array), `loading` (Boolean).
- **Events**: `@product-updated`, `@product-deleted`.
- **Methods**: `syncWithERPNext()`, `exportProducts()`.

### 2. خدمات الذكاء الاصطناعي (`ForecastService`)
نظام توقعات مدمج يعتمد على البيانات التاريخية.
- **الميزات**: كاش للنتائج (TTL 5 دقائق)، تنبيهات مخزون حرجة، وتوقعات موسمية.

### 3. تكامل Blogger (`BlogService`)
خدمة موحدة لجلب البيانات من Blogger لتقليل تكرار الكود في صفحات التصنيفات.

## ⚙️ إعداد المشروع
```bash
# تثبيت التبعيات
npm install

# تشغيل المشروع في بيئة التطوير
npm run serve

# بناء المشروع للإنتاج
npm run build
```

## 🔒 إعدادات البيئة
يرجى إنشاء ملف `.env` بناءً على `.env.example` وتعبئة المتغيرات التالية:
- `VUE_APP_ERPNEXT_URL`: رابط نظام ERPNext.
- `VUE_APP_EMAILJS_USER_ID`: معرف المستخدم في EmailJS.

## 🧪 الاختبارات
لتشغيل الاختبارات (Unit Tests):
```bash
npm run test:unit
```
