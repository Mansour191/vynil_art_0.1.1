## 🚀 خطوات سريعة للبدء

### 1️⃣ التحقق من المتطلبات

```bash
# التحقق من إصدارات Node.js
node --version  # يجب أن يكون >= 18

# التحقق من npm
npm --version   # يجب أن يكون >= 9

# التحقق من مساحة القرص
df -h         # يجب أن يكون متاح 2GB على الأقل
```

### 2️⃣ نقل الملفات بأمان

```bash
# 1. أخذ نسخة احتياطية
cp -r /path/to/current/frontend /backup/frontend-$(date +%Y%m%d)

# 2. نسخ الملفات الجديدة
cp -r /path/to/new/frontend/* /path/to/current/frontend/

# 3. نسخ الوثائق
cp -r docs/ /path/to/current/frontend/docs/
```

### 3️⃣ الإعداد والتشغيل

```bash
# 1. تثبيت الاعتماديات
npm install

# 2. إعدادات البيئة
cp .env.example .env
nano .env

# 3. البناء
npm run build

# 4. التشغيل
npm run dev
```
